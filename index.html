Démo GitHub Actions → Discord

Chaque push sur ce dépôt déclenche un workflow qui envoie un message vers un webhook Discord.

  • URL du webhook (de test) : discord.com/api/webhooks/…
  • Message envoyé : dépôt, branche, auteur du push, liste des commits.
  • Fichier du workflow : .github/workflows/discord.yml.

Modifiez ce fichier puis poussez : vous verrez le message arriver côté Discord.
